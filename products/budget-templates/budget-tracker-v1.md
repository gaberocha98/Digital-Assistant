## Layout per Budget Sheet

### Header Area
- Budget Period: customizable start/end dates
- Currency selector (default $)
- Spending summary bar (6 segments)
- Spent / Left to Spend overview

### Cash Flow Summary
| Row | Description |
|-----|-------------|
| + | Rollover (from previous period) |
| + | Income |
| - | Expenses |
| - | Bills |
| - | Savings |
| - | Debt |
| = | LEFT (remaining) |

### Income Section (30 rows)
- Category name | Planned $ | Actual $
- Sample: Salary, Self-Employment, Gig Earnings, Investment Interest, Social Media, Bonus

### Expenses Section (30 rows)
- Category | Planned $ | Actual $ | Progress %
- Sample: Groceries, Takeout, Restaurants, Household, Commuting, Fuel, Gifts, Cosmetics, Online Classes, Entertainment

### Savings Section (12 rows)
- Category | Planned $ | Actual $ | Progress %
- Sample: New Furniture, House Deposit, Driving Lessons, Christmas Fund, Vacation, New Laptop, School Supplies, Designer Bag

### Debt Section (13 rows)
- Checkbox (paid?) | Category | Planned $ | Actual $ | Progress %
- Sample: Mortgage, Car Lease, iPhone Financing, Apple Watch Loan, Credit Card

### Bills Section (30 rows)
- Checkbox (paid?) | Category | Planned $ | Actual $ | Progress %
- Sample: Housing, Electricity, Water, Cell Phone, Heating, Internet, Spotify, YouTube Premium, Netflix, HBO, Gym

### Category Sidebar Summary
- Total per category (Expenses, Bills, Savings, Debt)
- Planned vs Actual comparison chart
